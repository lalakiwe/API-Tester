API-Tester
==========

A simple HTTP Restful API test program made by QML. Only support GET and POST now.

**How to use**
 - Windows 7 64bit
	 - unzip win_runtime.zip
	 - run win.bat
 - Ubuntu 14.04 64bit
	 - $ qmlscene APITester.qml

**Develop environment**

 - Windows 7 64bit
	 - Visual Studio 2012
	 - QT 5.3.2 
	 - Qt Creator 3.2.2
 - Ubuntu 14.04 64bit
	 - gcc 4.8.2
	 - QT 5.3.2 (gcc 4.6.1)
	 - Qt Creator 3.2.2
